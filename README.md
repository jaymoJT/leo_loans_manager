# leo_loans_manager
leo_loans_manager is a system for individuals who offer loans and would want to digitize their management; 
   it registers clients ,
   disburse loans , 
   displays clients' loan history and summaries,
   Displays loan due date,
   Computes total loans disbursed and total loans paid back,
   Displays a summary of total unpaid loans,
   
  It's database file is tumainir_loan.sql...
  You will have to configure your database connection on the "partials/connection.php" file to your right settings then import "tumainir_loan.sql" 
  to get your database ready.
